# Джура — тест-чекліст перед здачею

Проходити на проді (todoforest-a.vercel.app): раз на телефоні (Chrome або
Safari), раз на десктопі (Chrome). Для чистого стану — інкогніто-вікно.
Позначай: ✅ ок / ❌ баг (і кинь Клоду опис + скріншот).

## 1. Онбординг і профіль

- [x] Перший візит (інкогніто): показується онбординг із 3 картками (Кажи / Обоз / Похід) ✅
- [x] «До діла, джуро!» → екран «Ставай до лав» ✅
- [x] «Пропустити — сам розберуся» → одразу застосунок; після перезавантаження онбординг більше не з'являється ✅
- [x] «Вписатись у реєстр» з ім'ям → тост «Вітаю в лавах, …», ім'я золотим у хедері ✅ (баг «завелика кнопка на моб» — виправлено, перевірити після деплою)
- [x] Кнопка «Вписатись у реєстр» неактивна, поки ім'я порожнє ✅
- [x] «Потім — спершу до справ» → застосунок, у хедері «Увійти» ✅
- [x] Тап по імені в хедері → екран редагування, зміна імені зберігається ✅
- [x] «Вийти з реєстру» → тост, у хедері знову «Увійти», задачі НЕ зникли ✅

## 2. Кажи (текстовий парсинг) — критично для здачі

- [x] «завтра на курінь їхати, шаблі поточити — терміново, у п'ятницю гінця слати» → 3 задачі: завтра = 2026-07-20, п'ятниця = 2026-07-24, «терміново» → пріоритет «горить» ✅
- [x] «сьогодні», «післязавтра», «у понеділок» → правильні дати відносно реальної (не вигадані) ✅
- [x] Час розпізнається: «завтра о 10 подзвонити в банк» → чип «завтра · 10:00» ✅
- [x] Беззмістовний текст («привіт як справи») → помилка «не розчув жодної задачі», нічого не додано ✅
- [x] Порожнє поле → кнопка «Джуро, розбери!» неактивна ✅
- [x] Під час розбору: спінер + «Слухаю, отамане…», текстове поле заблоковане ✅
- [x] Після розбору: перекидає в Обоз, тост «Джура записав задач: N», нові картки блимають малиновим ✅

## 3. Голос (Chrome/Safari; у Firefox мікрофона нема — це очікувано)

- [x] Тап мікрофона → браузер питає дозвіл → після дозволу кнопка пульсує ✅
- [x] Диктування українською → текст з'являється в полі наживо, під час паузи фіксується ✅
- [x] Повторний тап → запис зупиняється, текст лишається, можна редагувати руками ✅
- [x] Заборонити доступ до мікрофона → зрозуміле повідомлення (не мовчазна поломка) ✅
- [x] iPhone: мікрофон працює в Safari ✅

## 4. Обоз

- [x] Сортування: протерміновані зверху → сьогодні → майбутні за датою → без дати; у межах одного дня — «горить» перед «до діла» ✅
- [x] Тап по картці → відкривається редагування дати і пріоритету; зміни живуть після перезавантаження ✅ (+ додано редагування часу, перевірити після деплою)
- [x] Чекбокс → задача їде в «Виконано (N)» унизу; секція розгортається/згортається ✅
- [x] ✕ → тост «Задачу викреслено · Повернути»; «Повернути» повертає задачу на місце ✅
- [x] Тост зникає сам за ~5 секунд ✅
- [x] Порожній Обоз (новий юзер): екран «В обозі порожньо» з кнопкою в Кажи ✅

## 5. Похід

- [x] Задача з датою сьогодні → секція «Сьогодні», відсортовано за часом (без часу — внизу) ✅
- [x] Задача з минулою датою → секція «Протерміновані — наздогнати», червоний чип дати ✅
- [x] Виконана сьогоднішня задача → секція «Виконано сьогодні» ✅
- [x] Бейджі на вкладках Обоз/Похід відповідають реальній кількості невиконаних ✅
- [x] Порожній Похід: «Сьогодні походу нема» з кнопкою в Кажи ✅
